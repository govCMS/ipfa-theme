1.2.1Marked text #permalink

For highlighting a run of text due to its relevance in another context, use this tag.

Example
You can use the mark tag to highlight text. 
You can use the mark tag to <mark>highlight</mark> text.

1.2.2Deleted text #permalink

For indicating blocks of text that have been deleted use the tag.

Example
This line of text is meant to be treated as deleted text. 
<del>This line of text is meant to be treated as deleted text.</del>

1.2.3Underlined text #permalink

To underline text use the <u> tag.

Example
This line of text will render as underlined 
<u>This line of text will render as underlined</u>

1.2.4Small text #permalink

For de-emphasizing inline or blocks of text, use the <small> tag to set text at 80% the size of the parent. Heading elements receive their own font-size for nested <small> elements.

Example
This line of text is meant to be treated as fine print. 
<small>This line of text is meant to be treated as fine print.</small>

rendered as bold text

1.2.6Italic text #permalink

For emphasizing a snippet of text with italics.

rendered as italicized text

1.2.7Blockquote #permalink

For quoting blocks of content from another source within your document.

Example

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.

<blockquote>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
</blockquote>

1.2.8Subscript and super script #permalink

Subscript and super script

Example

Lorem ipsum sub dolor sit amet, super elit. Integer posuere erat a ante.

  <p>Lorem ipsum <sub>sub</sub> dolor sit amet, <sup>super</sup> elit. Integer posuere erat a ante.</p>

A list of items in which the order does not explicitly matter.
